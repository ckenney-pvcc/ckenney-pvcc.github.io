Term Definition
Network Classifications
  • LAN

  • MAN

  • WAN
 What computer networks are classified as (p.184)
  • Local area network consisting of several computers spanning a single building/building complex (p.184)

  • A metropolitan network that spans a local community (p.184)

  • A wide area network that spans a great distance such as neighboring cities (p.184)
Network models
  • bus network

  • star network
 The pattern in which the machines are connected (p.186)
  • bus network - When computers communicate directly with each other over a common bus (p.187)

  • star network - A central machine acts as a focal point and communicates with other machines using a radio broadcast
Protocols
  • CSMA/CD Ethernet

  • CSMA/CA Wifi
 The rules by which activities are conducted (p.187)
  • Carrier Sense, Multiple Access with Collision Detection controls the right to transmit messages in a bus network (p.187)

  • Carrier Sense, Multiple Access with Collision Avoidance is a protocol in which wireless systems avoid collisions instead of eliminating them completely (p.188)
Combining networks
  • Repeater

  • bridge

  • switch

  • router

  • gateway
 connecting networks to combine a larger version of the same type of network (p.191)
  • Passes signals between two original buses (p.191)

  • Connects two buses and forwards the message across the connection only when the message is meant for the other computer (p.191)

  • Connects several buses and considers the destination for each message and forwards the messages only to the computers that are part of its connection (p.191)

  • Connects two networks to forward messages (p.192)

  • The point where a network is linked to the internet (p.194)
Client/server model
  • P2P: peer-to-peer

  • distributed system

  • cloud computing
 Defines the roles of the processes as a client or a server. Clients make requests for other processes and a server satisfies those requests.
  • Processes that provide service and receive service from each other (p.194)

  • Software units that executed processes from other computers.

  • Shared computers are allocated for the use of clients (p.194)

  • Tier 1

  • Tier 2

  • Tier 3
 Assigns a unique address to each computer in the system for identification (p.202)

 internet-wide addressing system that gives an identifying address to each computer in the system (p.202)

  • DNS - Domain number system. A directory system that names servers makes up. (p.206)

 Connects devices to the internet.

 embedding hyperlinks into other documents transferring hypertext across the internet (p.206)

Cybersecurity
  • malware

  • virus

  • worm

  • Trojan horse

  • spyware

  • phishing

  • DDoS attack
Protecting computers, data, and networks from attacks (p.227)
  • Malicious software used for attacks (p.227)

  • Software that infects computers. It inserts itself into programs already in the machine and is executed when the host program executes (p.227)

  • A program that transfers itself through networks and copies itself to other computers (p.228)

  • A program that disguises itself as something enticing that the user willingly imports and then performs additional harmful activities (p.228)

  • Software that collects information about the activities on the computer and reports back (p.228)

  • Sends an email posing as a legitimate institution and solicits information from the user (p.228)

  • Denial of service. Not allowing the service of a computer to function (p.228)
